## Write a function to count the number of parameters passed to it.

### Instruction

Return the number of parameters passed.

### Example

For this input

```
args = 6, 23, 69, 22, 9
```

the result should be:

```
5
```

**Reason:** The function received **5** params: **6**, **23**, **69**, **22**, and **9**.  
Therefore, the expected output is **5**.

### Example

For this input

```
args = 'j', 'k'
```

the result should be:

```
2
```

**Reason:** The function received **2** params: `'j'` and `'k'`.  
Therefore, the expected output is **2**.
