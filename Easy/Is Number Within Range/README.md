## Write a function to check if a number is within a given range.

### Instructions

If the number `num` is within the range of `lowerLimit` (inclusive) and `upperLimit` (inclusive), return `"Yes"`. Otherwise, return `"No"`.

### Example

For this input

```
num = 10
lowerLimit = 1
upperLimit = 10
```

the result should be:

```
"Yes"
```

**Reason: 10** is within the range of **1** and **10**, so the output is `"Yes"`.
