## Write a function to check if a string contains spaces.

### Instructions

If the string `str` contains space, return `"Space"`. Otherwise, return `"No Space"`.

### Example

For this input

```
str = "Hello World"
```

the result should be:

```
"Space"
```

**Reason:** The string `"Hello World"` has a space between `"Hello"` and `"World"`, so id does contain a space.
