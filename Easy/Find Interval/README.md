## Write a function to find the interval of an array.

### Instructions

Interval is the difference between the largest number and the smallest number in an array.

For example, the interval of the array `[10, 5, 15]` is: **15 - 5 = 10**.

Return the interval of the `arr` array.

### Example

For this input

```
arr[] = [5, 2, 9, 1]
```

the result should be:

```
8
```

**Reason:** The maximum number in the array is **9** and the minimum number is **1**. So, the interval is **8**.

### Example

For this input

```
arr[] = [7]
```

the result should be:

```
0
```

**Reason:** The maximum number in the array is **7** and the minimum number is **7**. So, the interval is **0**.
