## Write a function to check if two sublists are identical.

### Instructions

Two subsets are said to be identical if they have the same elements in the same order. For example,

1. `[2, 5]` and `[2, 5]` are identical lists.
2. `[3, 1]` and `[1, 3]` are not identical lists.

If the two sublists `"list1"` and `"list2"` are identical, return `"Identical"`. Otherwise, return `"Not Identical"`.

### Example

For this input

```
list1[] = [1, 2, 3]
list2[] = [1, 2, 3]
```

the result should be:

```
"Identical"
```

**Reason:** Both lists have the same elements in the same color: **1**, **2**, and **3**. So they are identical.
