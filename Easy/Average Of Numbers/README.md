## Write a function to calculate the average of all the numbers in an array.

### Instructions

The average of numbers is calculated by summing them up and dividing by the total count of numbers.

For example, the average of numbers **10**, **20** and **30** is `(10 + 20 + 30) / 3 = 20`.

Return the average of all numbers in the array `arr`.

### Example

For this input

```
arr[] = [4, 6, 8, 10]
```

the result should be:

```
7
```

**Reason:** The average of the numbers **4**, **6**, **8**, and **10** is `(4 + 6 + 8 + 10) / 4 = 7`.
