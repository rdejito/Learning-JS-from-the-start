## Write a function to make the sum of two numbers even.

### Instructions

Return the sum of `num1` and `num2`. If the sum is not even, add 1 to it.

### Example

For this input

```
num1 = 4
num2 = 3
```

the result should be:

```
8
```

**Reason:** The sum of **4** and **3** is **7**, which is not even. Adding **1** makes it even, so the result is **8**.
