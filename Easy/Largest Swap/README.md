## Write a function to perform the largest swap.

### Instructions

Given a two-digit number, determine if it is larger than the number formed by swapping its digits.

For example, swapping the digits of **42** results in **24**. Here, **42** is the largest number between the two.

If a number `num` is already the highest possible, return `true`.  
Otherwise, return `false`.

### Example

For this input

```
num = 27
```

the result should be:

```
false
```

**Reason: 27** is not the highest possible number. By swapping the digits in **27**, we get **72**, which is greater than **27**.
