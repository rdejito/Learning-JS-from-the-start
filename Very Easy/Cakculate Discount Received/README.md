## Write a functin to calculate the discount received.

### Instructions

We can find the discount received by subtracting the discounted price from the original price.
For example, `given originalPrice = 200` and `discountedPrice = 150`, the discount received is `200 - 150 = 50`.
Return the discount received from the given original price and discounted price.

### Example

For this input

```
originalPrice = 500
discountedPrice = 400
```

the result shoud be:

```
100
```

**Reason:** The discount received is `500 - 400 = 100`.
