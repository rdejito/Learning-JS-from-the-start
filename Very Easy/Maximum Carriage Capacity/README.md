## Write a function to calculate the maximum number of people a carriage can hold.

### Instructions

The maximum number of people that can fit in the carriage is given by `carriageArea / personArea`.

Return the maximum number of people the carriage can hold for given two integers `carriageArea` and `personArea`.

### Example

For this input

```
carriageArea = 100
personArea = 5
```

the result should be:

```
20
```

**Reason: 100** divided by **5** equals **20**. So, a carraige with an area of **100**, where each person occupies an area of **5**, can hold up to **20** people.
