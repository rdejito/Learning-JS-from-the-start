# Round Off a Number

---

## Instructions

Rounding off a number means adjusting it to the nearest whole number or specified decimal place for simplicity.

For example:

1. Rounding off `8.08741565` to `0` decimal places results in `8`.
2. Rounding it to `3` decimal places results in `8.087`.

   Hint: Use the toFixed() builtin method to round off.

## Example

For this input

```
num = 0.778899
ndigits = 2
```

the result should be:

```
"0.78"
```

**Reason**: Rounding off `0.778899` to `2` decimal places results in `"0.78"`.
