## Write a function to check if the sum of two numbers is less than the third number.

### Instructions

If the sum of `num1` and `num2`, return `true`.  
Otherwise, return `false`.

### Example

For this input

```
num1 = 5
num2 = 7
num3 = 15
```

the result should be:

```
true
```

**Reason:** The sum of **5** and **7**, which is **12**, is less than the third number, which is **15**. So, it returns `true`.
