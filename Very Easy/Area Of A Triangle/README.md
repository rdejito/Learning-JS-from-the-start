## Write a function to calculate the area of a triangle.

### Instructions

Return the area of a triangle with base `base` and height `height`.

**Hint:** The formula for area of triangle is:
`area = 0.5 * base * height`.

### Example

For this input

```
base = 10
height = 5
```

the result should be:

```
25
```

**Reason:** The area of a triangle with a base of **10** and a height of **5** is **25**.
