## Write a function to calculate the cube of a number.

### Instructions

Given an integer `num`, return its cube.

**Hint:** The cube of a number is obtained by multiplying itself three times.

For example, the cube of 3 is 27 because `3 * 3 * 3 = 27`.

### Example

For this input

```
num = 4
```

the result should be:

```
64
```

**Reason:** The cube of **4** is `4 * 4 * 4`, which equals **64**.
