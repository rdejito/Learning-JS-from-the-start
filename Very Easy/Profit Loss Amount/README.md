## Write a function to calculate the profit or loss amount.

### Instructions

The profit or loss amount is calculated by subtracting the cost price from the selling price:

```
Profit/Loss = selling price - cost price
```

Calculate the amount and return:

1. `"Profit"`if the is positive.
2. `"Loss"` if the result is negative/
3. `"Break-even"` if the result is zero.

### Example

For this input

```
sellingPrice = 50
costPrice = 30
```

the result should be:

```
"Profit"
```

**Reason:** Subtracting cost price **150** from selling price **100** gives us a loss of **50**.
