## Write a function to find the largest of two numbers.

### Instructions

Return the larger number between `num1` and `num2`.

**Note:** Simple return the number if both numbers are equal.

### Example

For this input

```
num1 = 12
num2 = 8
```

the result should be:

```
12
```

**Reason: 12** is greater than 8. So, the largest number is 12.
