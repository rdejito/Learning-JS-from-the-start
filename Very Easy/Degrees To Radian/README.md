## Write a function to convert an angle given in degrees t radians.

### Instructions

An angle can be converted to radians using the formula `radians = (pi/180) * degrees`.

for example, given `degrees = 270`, converting it to radians results in `radians = (pi/180) * 270 = 4.712`.

Return the converted value.

### Example

For this input

```
degrees = 90
```

the result should be:

```
1.571
```

Reason: 90 degrees is equivalent to 1.571 radians.
