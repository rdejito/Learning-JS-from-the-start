## Write a function to convert a number into string with dashes between each digit.

### Instructions

Consider the number 25468. Inserting dashes between each digit would result in `"2-5-4-6-8"`.

Insert dashes `-` between each digit and return the newly formed string.

### Example

For this input

```
num = 12345
```

the result should be:

```
"1-2-3-4-5"
```

Reason: Putting dashes between each digit in 12345 results "1-2-3-4-5".
