## Write a function to add even numbers from 1 to n (inclusive).

### Instructions

Return the sum of all even numbers from 1 to `n` (inclusive) in the given integer `n`.

## Example

For this input

```
n = 10
```

the result should be:

```
30
```

Reason: The sum of all even numbers from 1 to 10 is 30 `( 2 + 4 + 6 + 8 + 10 = 30)`.
