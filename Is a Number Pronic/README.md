## Write a function to check if a number is pronic.

### Instructions

A pronic number is a product of two consecutive integers.

For example, **42** is a pronic number because it is obtained by multiplying two consecutive integers **6** and **7** i.e, `6 * 7 = 42`.

Return `"Pronic"` if the given integer is a pronic number.
Otherwise, return `"Not Pronic"`.

### Example

For this input

```
num = 12
```

ther result should be:

```
"Pronic"
```

**Reason: 12** can be expressed as **3 * 4**, where **3** and **4** are consecutive numbers. So, it's a pronic number.
