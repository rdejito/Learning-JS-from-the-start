## Write a function to replace smiley faces ":)" with sad faces ":(".

### Instructions

Convert all occurences of `":)"` to `":("` in the given string and return the new string.

### Example

For this input

```
input = ":) :) :) All smiles here :)"
```

the result should be:

```
":( :( :( All smiles here :("
```

Reason: Replacing `:)` with `:(` in `":) :) :) All smiles here :)"` results `":( :( :( All smiles here :("`.
