## Write a function to find the largest number that can be formed by sqapping two digits of a given number.

### Instructions

Suppose, the given is **39452**, if we swap two digits 3 and 9, we get **93452**, which is the largest number you can form by swapping any two digits in **39452**.

Return the largest possible number by swapping two digits in the given int `num`.

### Example

For this input

```
num = 2736
```

the result should be:

```
7236
```

**Reason:** Swapping **2** and **7** results in **7236**, which is the largest possible number that can be formed by swapping two digits of **2736**.
